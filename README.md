<!-- Run Locally -->
### :running: Run Locally

Clone the project

```
  git clone https://github.com/jorofino3/portfolio
```

Go to the project directory

```
  cd app
```

Install dependencies

```
  npm install
```

Start the server

```
  npm start
```
